THEMES

Arrow Keys:

User can click on the arrow keys to move the map

Zoom Fit:

User can click on the button to zoom fit the map

Zoom In:

User can click on the button to zoom in

Zoom Out:

User can click on the button to zoom out

Address Finder:

User can click on the button and the map twice to display the address

Time:

User can click on the button twice to display the time

Map Loader:

User can select a location from the drop down and click the button to load that location

Intersection Search:

User can type in two streets and click the button to find the corresponding intersection

Color Themes:

User can select a color theme based on their preference from the drop down and by clicking the button
