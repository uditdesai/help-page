THEMES

Desai and Kooresh Maps is a global mapping system for select locations around the world. It was created for a software programming course at the University of Toronto.

The maps are designed for the user in mind making sure the user is not cluttered with data and all features are easy to use and see. Using the maps, you can find different places in any of the locations including point of interests, parks and even exact addresses of buildings.
